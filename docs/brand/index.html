FlexiWires

The design system

One source of truth for how FlexiWires looks, reads and behaves. One signature green, shared with the landing page, and a neutral grey ramp underneath it. Type is shared with the landing too. Every token is built to scale into the full website, and each rule carries its reasoning.

Version 2.0 · Updated June 2026
01 · Overview

Engineered, calm, confident

FlexiWires compares every cross-border rail on one screen. The system is precise and quiet, with one decisive color: a signature green that marks both the next action and the win. These three traits drive every decision below.

Engineered

Monospace data, exact figures, tight grids. Built by people who move money.

Calm

A neutral canvas and generous space. Green is rationed so it always means something.

Confident

One green for action and outcome. Where it appears, something matters.

One color, shared

The system uses the same signature green and the same type as the live landing page, so marketing and product read as one product. Green is the only chromatic color; everything else is a neutral grey. There is no second brand color.

02 · Color

One signature green

Green leads and a neutral grey ramp holds it on a near-white canvas. Green is rationed to action, status and the positive outcome, so every green pixel stays meaningful. It is always a fill or a small accent paired with dark text, never a text color on white.

Why one green

Signature green (#3FE864) reads as go, success and money saved, exactly the feeling at send. Rationing it to CTAs, status dots and the live indicator keeps it meaningful, and black-on-green stays AAA legible.

Green is light, so it only ever carries dark text and never appears as text on a white background. When green needs to be a text or thin-line accent on light, use Green Deep (#1C8F3E), which is verified for contrast. The neutral grey ramp on a near-white canvas keeps the UI calm and lets green do all the talking.

03 · Accessibility

Contrast, validated

Every text and surface pairing is checked against WCAG. Green carries dark text; the neutrals carry ink. These are the approved combinations.

The rule

Body text needs 4.5:1 (AA); large text and UI 3:1. Never put white text on green or green text on white, both fail. Green is a fill paired with ink; for green-toned text or icons on light, switch to Green Deep.

04 · Typography

Three typefaces, one scale

Shared with the landing page. A neutral workhorse for everything, an expressive serif for one accent line, and a precise mono for data. Each role maps to a need a treasury product has: scanning, trust and emotion.

05 · Spacing & layout

A 4px rhythm

All spacing steps from a 4px base. Most layout uses 16 / 24 / 32; sections breathe at 64 / 96. Sticking to the scale keeps vertical rhythm consistent across product and marketing.

One ramp means any two components placed together already agree on spacing. It removes per-screen guesswork and keeps density even as the product grows.

Border-darken, never shadow-lift

On hover, cards darken their border rather than floating up on a big shadow. It reads as precise and engineered, not bouncy, and avoids the heavy drop-shadows that make a finance UI feel cheap.

07 · Iconography

Line icons, one weight

Thin geometric line icons (Lucide family), never filled, never multi-color. Stroke width scales gently with size so icons read consistent everywhere.

One green action per screen

The single green button names the next valuable action (start, confirm, send). Everything else is ghost. If two buttons compete for green, neither feels decisive, so there is only ever one.

09 · Status & feedback

One dot says live

Status uses color plus a label, never color alone. Green means live, best or positive; amber means attention.

Live Best rate Settled Pending
Why color plus label

Roughly 1 in 12 men cannot reliably separate red and green. A word next to the dot keeps every status legible to everyone, and keeps meaning intact in screenshots and print.

10 · Motion

Restrained, purposeful

Motion confirms the product is live and rewards interaction without pulling focus from the task. It always respects prefers-reduced-motion.

Live corridor rail

An always-moving marquee of corridors signals an active, real-time network.

Reveal on scroll

A gentle fade and rise as sections enter. Subtle, never a slide-show.

Border on hover

Cards shift their border color instead of leaping on a shadow. Precise, not bouncy.

11 · Principles

How it all composes

The tokens above only matter if they are applied the same way every time. These are the habits that keep FlexiWires feeling like one engineered product.

Do

  • Ration green; let neutrals carry the page
  • Use mono for every number and label
  • Separate surfaces with border first, shadow second
  • Pair status color with a word
  • Step spacing and radius from the scale

Avoid

  • Green text on white, or white text on green
  • A second accent color competing with green
  • Heavy drop-shadows or lift-on-hover
  • Serif anywhere but the one accent line
  • Color as the only signal of meaning
